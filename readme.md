# 🚚 TransitOps

TransitOps is a modern transport operations platform designed to simplify fleet management for logistics organizations. It provides a centralized system to manage vehicles, drivers, trips, maintenance, fuel logs, and operational expenses while enforcing business rules such as driver eligibility, vehicle availability, and cargo capacity validation.

## ✨ Features

- 🔐 Secure Authentication & Role-Based Access Control (RBAC)
- 🚛 Vehicle Registration & Fleet Management
- 👨‍✈️ Driver Management with License Tracking
- 🗺️ Trip Creation, Dispatch & Lifecycle Management
- 🔧 Vehicle Maintenance Management
- ⛽ Fuel Log & Expense Tracking
- 📊 Dashboard with Fleet KPIs and Operational Insights
- 📈 Reports for Fuel Efficiency, Fleet Utilization, and Operational Costs

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL with Prisma ORM
- **Authentication:** JWT & bcrypt

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Generate Prisma Client
npx prisma generate

# Run database migrations
npx prisma migrate dev

# Start the development server
npm run dev
```
